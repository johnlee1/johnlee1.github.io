Ana
Ana is a web application that allows users to collaboratively comment on a text. The frontend application was built with React, Draft.js, Material UI, and Socket.io. The draft version of the React app was hosted on AWS S3. The backend application was built with Node.js, Express.js, and Socket.io. It was hosted on an AWS server. Check out my github account (johnlee1) to see the UI code and the server code.

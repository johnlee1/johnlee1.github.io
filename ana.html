Ana
Ana is a web application that allows users to collaboratively comment on a text. A first draft of the app is viewable at http://anaui.s3-website.us-east-2.amazonaws.com/. The frontend application is being built with React, Draft.js, Material UI, and Socket.io. The draft version of the React app is being hosted on AWS S3. The backend application is being built with Node.js, Express.js, and Socket.io. It is currently being hosted on an AWS server. This is an ongoing project. Check out my github account (johnlee1) to see the UI code and the server code.

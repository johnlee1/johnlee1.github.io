the tech stack

The frontend is an angular 2 app served by nginx. The api backend is a nodejs app using hapi. Both nginx and node live in docker containers that are deployed on aws ec2 and configured by ecs. The nodejs app is connected to a mongodb database provided by mlab.



looking forward

design: A style guide, vision guide, and prayer guide are all in the process of being created in order to establish the intended ux of the app which will inform how the ui will be built.

security: More filtering and validation needs to be added on both the client and server.

development: I also want to look into options for integrating CI/CD into the development workflow.

infrastructure: Currently the project is set up in a way that does not allow the frontend and backend to scale independently of each other since they are both configured to live on the same server instance. In the future it could be beneficial to have a more flexible microservices setup that allows independent horizontal scaling.
